# PortMidi Test

A simple test application that uses
[RtMidi](https://www.music.mcgill.ca/~gary/rtmidi/) and the
[`go-rtmidi`](https://pkg.go.dev/github.com/mattrtaylor/go-rtmidi) package.
``` sh
go get github.com/mattrtaylor/go-rtmidi
```

# Compiling

``` sh
make
```

# Running

``` sh
./kronut
```
